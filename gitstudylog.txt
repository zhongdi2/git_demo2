git init 初始化仓库
git status 查看当前状态
git add 将文件添加到暂存区
git commit -m "描述" 将文件提交到仓库
git reset HEAD   <file>  撤出暂存区文件
git checkout -- <file>..." 丢弃工作区的改动

版本回滚：
git log 查看提交历史

git reset --hard  回滚版本

git rm 将本地文件清空 清空之后要进行  git commit -m

标签管理
git tag 列出所有标签
git tag name 创建标签
git push origin v1.0 将标签推送到远程
git tag   -d name  删除标签

git checkout 切换分支
git merge 合并分支
